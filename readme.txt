Deposited "renormalizer.zip" is the version of "Renormalizer" code used for simulations in  the paper " A numerically exact description of ultrafast vibrational decoherence in vibration-coupled electron transfer".
For the latest version of "Renormalizer" code and documents, please check "https://github.com/shuaigroup/Renormalizer".

"ET-H.py" is a test input file.
The folder "par" contains the phsyical data for the test.
